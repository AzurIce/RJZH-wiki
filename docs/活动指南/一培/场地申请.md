# 场地申请

> 参考 [[场地申请指南#一、学活场地申请#1.2 学活小剧场申请]]

一般申请 **学活小剧场（302）**，卡点抢场地，然后提前跑场地（不要到两三天的时候还没跑）。

## 场地信息填写

- **一培场地信息：**
	 - **活动主题**：
		 软件学院新生辩论赛“xx杯”启动仪式暨第一次培训活动
	- **活动内容**（当然你可以自己憋一个不这么糊弄的）：
		通过表演，讲解等方式对新生进行培训，新生在过程中可以了解辩论，辩论队。
	- **活动类型**：
		表演、培训
